HTS JSON lookup (2020 revision 14)

Tool to decipher HTS code by:

  1. Locate "searched" HTS number and it's parents. (indicated by indent)
  1. If footnotes from step 1 direct users to another HTS code, locate directed figure.
  1. If figure from step 2 guides user to a note for exclusions, locate the exclusions from note that matches the "searched" HTS number.
